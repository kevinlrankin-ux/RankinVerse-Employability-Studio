# RankinVerse Employability Studio

RankinVerse Employability Studio is a standalone Google Workspace-delivered course platform designed to prepare students for workplace entry, workplace conduct, safety, reporting, accountability, and early leadership.

## Delivery model

- Google Site front end
- Google Apps Script runtime
- Google Sheets for roster, progress, and settings
- Google Drive for student workspace provisioning
- Google Docs / Slides / Forms for learning assets
